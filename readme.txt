== Description ==
An Admin Tool plugin for bulk deletions. Works with BuddyPress and the BuddyBoss Platform.

It allows admins to bulk delete Activity, Group Activity, Message and Notification entries & meta based on an 'older than' date.

Adds a link under *Tools* menu.

Note: for Messages and Notifications, it will delete **BOTH** read and unread entries.

There are action hooks for pre and post deletion for all four entry types.

For more plugins, please visit [PhiloPress](https://www.philopress.com/)
